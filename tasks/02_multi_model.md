
# タスク02：別モデル同時表示機能の実装指示書

## 1. 目的
現在同じモデルしか表示できない4分割（Multi-View）機能を拡張し、イトカワとリュウグウなど、異なる小惑星モデルを同時に並べて形状や特徴をダイナミックに比較できるようにする。

## 2. 着手手順・手がかり
1. **バックアップの作成**:
   修正前に `src/aigis/` 内の関連フォルダを削除せず別名フォルダに変更・退避する。
2. **コードの修正**:
   `MainFrame.java` 内の `GLSplitWindow window` およびモデル読み込み処理（`.obj`ファイルのロード部分）を解析する。
   現在単一のモデルインスタンスを共有している箇所を、ビューポートのインデックス番号（0〜3）に応じて異なるオブジェクトを保持できるように拡張する。

## 3. 完了条件
- 異なる小惑星の `.obj` ファイルを別々の画面に同時に描画できるデータ構造への書き換えが完了していること。
- ビルドが正常に通り、差分が提示されていること。

【セッション1】解説表示・基本機能追加
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 解説パネル追加
- spectrumInfoPanel を MainFrame に追加
- loadDescription(key) メソッド実装
  → Description/{lang}/{key}.html をローカル優先でロード
  → JAR内リソースにフォールバック
- モデル切り替え時も解説パネルを維持表示

■ 日本語 MapData ラベル翻訳
- loadMapDataLabels() メソッド実装
  → Description/mapdata_labels_ja.properties を読み込み
- MapTableModel.getKeyAt() 追加 / getValueAt() を翻訳対応に変更
- イトカワ用 mapdata_labels_ja.properties 作成
    #None=なし（天体基本情報）
    AMICA_brightness=AMICA輝度（カメラ明るさ）
    Itokawa_elevation_64=標高
    itokawa_Gpotential20110111=重力ポテンシャル高度
    itokawa_slope20110111=重力斜度
    st_2422543206_v_map512=テクスチャマップ
- リュウグウ用 mapdata_labels_ja.properties 作成
    Gravitational_Slope / _3.50 / _4.00 / _7.63=重力斜度（各種）
    Acceleration=重力加速度
    Geopotential_Height=重力ポテンシャル高度

■ HTML解説ファイル作成（全データセット・日英）
- イトカワ: #None / AMICA_brightness / Itokawa_elevation_64
           itokawa_Gpotential20110111 / itokawa_slope20110111
           st_2422543206_v_map512
- リュウグウ: #None / Gravitational_Slope / _3.50 / _4.00 / _7.63
             Acceleration / Geopotential_Height

■ 起動時言語選択ダイアログ
- MainFrame static ブロックで locale を super() より前に設定
- 初回起動時: 日本語/English 選択ダイアログ表示
- 2回目以降: AiGIS.properties に保存した設定を自動適用
- 設定メニューの言語変更も AiGIS.properties に保存

■ HTML解説から「中学生向け」削除
- 全 ja/ HTMLの「かんたん説明（中学生向け）」→「かんたん説明」

【セッション2】バグ修正・出典追加・改善
━━━━━━━━━━━━━━━━━━━━━━

■ 起動ファイル: コンソールなし版追加
- 起動（コンソールなし）.vbs を新規作成
  → javaw.exe 使用でコマンドプロンプトなし起動

■ 解説表示バグ修正（MapData間切り替え）
- loadDescription() の JEditorPane.setPage() → setText() に変更
- readFileAsString() / readUrlAsString() ヘルパー追加
- 原因: setPage() は同一 JEditorPane への連続呼び出しで
        ページが更新されないことがある
- 修正後: Elevation→AMICA_brightness 等の切り替えで
        解説が消えなくなった

■ HTML解説に出典追加（日英）
- イトカワ: AMICA_brightness / Itokawa_elevation_64
           itokawa_Gpotential20110111 / itokawa_slope20110111
  → Saito+2006 / Fujiwara+2006 / Demura+2006 / Scheeres+2006 等
- リュウグウ: Gravitational_Slope（全4種）/ Acceleration
             Geopotential_Height
  → Watanabe+2019 / Scheeres+2019 / Sugita+2019 等
